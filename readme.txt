Il progetto è stato realizzato come applicazione web per servlet container Tomcat 8.5.12

Per la programmazione è stato utilizzato l'IDE Netbeans 8.1 con JDK 1.8

Per poter provare l'applicazione è sufficiente avere a disposizione un server Tomcat e, una
 volta aperto Netbeans, dare il comando per compilare l'intero progetto.
Completata la compilazione e l'avvio di Tomcat, se l'IDE è configurato appositamente, in
automatico verrà aperto il browser all'index che è stato appositamente creato per gestire i
vari step.

A questo punto è sufficiente utilizzare le pagine web per provare i vari step dell'esercizio.

Dal quinto step in poi, nel caso in cui vengano inseriti numeri negativi viene mostrato
il messaggio di error richiesto dalla consegna; per poter ritornare al form e necessario utilizzare
il comando indietro del browser oppure reiserire l'indirizzo dell'Index.



La configurazione del mio PC prevedeva l'apertura del browser all'indirizzo:
	
	http://localhost:8080/StringCalculator/

per la visualizzazione della pagina Index.html
